# 🛡️ AI SMS Guardian

An AI-powered SMS spam detection web application that detects spam and phishing messages using a fine-tuned DistilBERT model and provides explainable AI analysis using Groq Llama 3.1.

## 🚀 Live Demo

🔗 https://ai-sms-guardian-j3zben22e5u2bnvetkvaub.streamlit.app/


## 📌 Features

- 🤖 AI-powered SMS Spam Detection
- 🧠 Fine-tuned DistilBERT Model
- 💡 Explainable AI using Groq Llama 3.1
- 📊 Risk Score & Threat Level
- 🚨 Scam Type Detection
- 📄 Downloadable PDF Report
- 🌐 Deployed on Streamlit Cloud
- 🤗 Model hosted on Hugging Face

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/AashuOne8/AI-SMS-Guardian.git
```

Go to the project folder

```bash
cd AI-SMS-Guardian
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
